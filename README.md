<h1> form validation for absolute beginners</h1>
<p> wanna practice or see how form validates ? well you are in the right place</p></br>
<p>the logic is simple-</p>
<ol>
  <li>add an event listener on the submit button.</li>
  <li>after that, check if any of the input is empty ( input.value.trim()==="" ).</li>
  <li>if yes, create an alert or create a container to contain the error message and write the error message with javascript using "innerText"</li>
</ol>
